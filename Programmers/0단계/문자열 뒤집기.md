# 소스코드
```
class Solution {
    public String solution(String my_string) {
        StringBuffer sb = new StringBuffer(my_string);
        String answer = sb.reverse().toString();
        return answer;
        
    }
}
```
# 풀이
- ![StringBuffer 클래스]()의 메소드인 reverse()와 ![String 클래스]()의 메소드인 toString()을 사용
